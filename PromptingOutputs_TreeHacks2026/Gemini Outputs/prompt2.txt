Input: You are an expert OpenSCAD CAD modeler. Generate an OpenSCAD script that creates a 3D model of a classic American fire hydrant matching the attached reference image (front-facing studio photo). The model should be clean, printable, and parameterized.



GOALS

- Output ONLY OpenSCAD code (no explanations unless as code comments).

- Use parametric design: key dimensions as variables at the top.

- Use solid primitives (cylinder, sphere, cube) + boolean ops (union, difference, hull, minkowski if needed).

- Favor printable geometry: manifold, no self-intersections, avoid paper-thin walls.

- Add reasonable fillets/chamfers using minkowski with small spheres/cylinders or approximate with hulls where appropriate (light rounding, not overkill).

- Center the model at the origin with the base sitting on Z=0.

- Provide a final union() named module hydrant(); and call it at end.



VISUAL FEATURES TO MATCH (from the image)

1) Base: wide circular flange at the bottom with a thicker rim and bolt bumps (or simplified bolt heads) evenly spaced around.

2) Main body: vertical cylindrical column with subtle step changes (slight taper/shoulders) and a thicker mid-band.

3) Side nozzles: two symmetric side outlets (left and right), each a short cylindrical protrusion with a cap-like end. Include a small rectangular lug/boss on each nozzle (as seen).

4) Front feature: a small circular boss on the front midsection, plus a short conical/rounded protrusion on it (like a screw/bump).

5) Top: domed/helmet cap with vertical ribbing/grooves and a small top nut/knob (dark cap in photo). Include the seam line or step where the dome meets the body.

6) Chains: two short chains draping from the left and right nozzle areas to the front (center) area. These can be approximated: either (A) simplified as smooth catenary-like tube segments, or (B) as linked torus-like chain links (lower poly is fine). Make them optional via a parameter include_chains=true.



DIMENSIONS (approximate; make adjustable)

- Overall height: ~200 mm

- Base diameter: ~110 mm

- Main body diameter: ~60–70 mm

- Dome max diameter: ~75–85 mm

- Side nozzle outer diameter: ~35–40 mm, length ~40–50 mm

- Front boss diameter: ~25–30 mm



MODEL STRUCTURE (suggested modules)

- module base_flange()

- module main_body()

- module dome_cap()

- module side_nozzle(sign=1) // sign=+1 right, -1 left

- module front_boss()

- module chain_pair() // optional

- module hydrant() // union of all



DETAILING REQUIREMENTS

- Add step transitions (shoulders) between base and body and between body and dome.

- Add dome ribbing: create ~10–16 shallow vertical ribs using rotate_extrude? (No) Prefer: subtract narrow rotated rectangular prisms around the dome OR add ribs by union of thin cylinders/boxes rotated around Z.

- Add bolt heads on the base ring: e.g., 6–8 evenly spaced small cylinders.

- Ensure symmetry: nozzles mirrored perfectly.



PRINTABILITY

- Avoid extremely thin ribs; minimum feature thickness ~1.5–2 mm.

- Chains: if include_chains=true, make them thicker and simplified so they print.



OUTPUT

- Provide a single OpenSCAD file.

- At top: a parameter section with comments.

- Use $fn values appropriate for smoothness but not huge (e.g., $fn=64 default).

- Finally call hydrant();



If you need to simplify, prioritize the overall silhouette and the dome ribbing + nozzles + base flange to match the reference image.

    Model based on reference image image_5a10c2.png

// --- PARAMETERS ---
$fn = 64;

// Overall Dimensions
overall_height = 200;
base_diameter = 110;
body_diameter = 65;
dome_diameter = 80;

// Component Dimensions
base_height = 12;
nozzle_diameter = 38;
nozzle_extension = 42;
front_boss_diameter = 30;
bolt_count = 8;
rib_count = 12;

// Options
include_chains = true; // Set to false for cleaner FDM printing
chain_thickness = 2.5;

// --- MODULES ---

module bolt_heads() {
    for (i = [0 : bolt_count - 1]) {
        rotate([0, 0, i * (360 / bolt_count)])
        translate([base_diameter / 2 - 8, 0, base_height])
        cylinder(h = 4, d = 8, $fn = 6); // Hexagonal bolts
    }
}

module base_flange() {
    union() {
        // Bottom Plate
        cylinder(h = base_height, d = base_diameter);
        // Step transition to body
        translate([0, 0, base_height])
        cylinder(h = 5, d1 = base_diameter * 0.85, d2 = body_diameter);
        bolt_heads();
    }
}

module main_body() {
    union() {
        // Lower trunk
        cylinder(h = 130, d = body_diameter);
        
        // Mid-section band (thicker area where nozzles attach)
        translate([0, 0, 80])
        difference() {
            cylinder(h = 45, d = body_diameter + 8);
            // Slight taper on the band edges
            translate([0,0,-1]) cylinder(h = 4, d1 = body_diameter + 10, d2 = body_diameter + 8);
        }
    }
}

module side_nozzle(side = 1) {
    rotate([0, 0, side * 90])
    translate([body_diameter / 2, 0, 105]) 
    rotate([0, 90, 0])
    union() {
        // Nozzle neck
        cylinder(h = nozzle_extension - 10, d = nozzle_diameter);
        
        // Nozzle cap
        translate([0, 0, nozzle_extension - 10]) {
            cylinder(h = 10, d = nozzle_diameter + 4);
            // Nut/Lug on the cap
            translate([0, 0, 10])
            cylinder(h = 6, d = 12, $fn = 6);
        }
        
        // Chain attachment loop (simplified)
        translate([nozzle_diameter/2, 0, 5])
        rotate([0, 90, 0])
        torus(r1=4, r2=1.5);
    }
}

module front_boss() {
    translate([0, -body_diameter / 2 - 2, 105])
    rotate([-90, 0, 0])
    union() {
        cylinder(h = 8, d = front_boss_diameter);
        // Center bump/screw
        translate([0, 0, 8])
        cylinder(h = 5, d1 = 12, d2 = 8);
        
        // Chain attachment center
        if(include_chains) {
            translate([0, 0, 2])
            rotate([90, 0, 0])
            torus(r1=3, r2=1.5);
        }
    }
}

module dome_cap() {
    translate([0, 0, 155])
    union() {
        // Lower rim of the dome
        cylinder(h = 10, d = dome_diameter);
        
        // The dome itself
        translate([0, 0, 10])
        difference() {
            sphere(d = dome_diameter);
            // Cut off bottom half of sphere
            translate([0, 0, -dome_diameter / 2])
            cube([dome_diameter + 1, dome_diameter + 1, dome_diameter], center = true);
            
            // Ribbing (subtractive grooves)
            for (i = [0 : rib_count - 1]) {
                rotate([0, 0, i * (360 / rib_count)])
                translate([dome_diameter / 2 - 2, 0, 15])
                rotate([0, 15, 0])
                cube([6, 3, 40], center = true);
            }
        }
        
        // Top Nut (dark cap in photo)
        translate([0, 0, 38])
        cylinder(h = 10, d = 18, $fn = 5); // Pentagonal or Hexagonal nut
    }
}

module torus(r1, r2) {
    rotate_extrude() translate([r1, 0, 0]) circle(r = r2);
}

module chain_pair() {
    if (include_chains) {
        // Simplified catenary approximation using a torus segment or hull
        // Loop from Left Nozzle to Center
        color("silver")
        translate([-20, -25, 100])
        rotate([0, 40, -45])
        scale([1, 1, 1.5])
        torus(r1=25, r2=chain_thickness/2);

        // Loop from Right Nozzle to Center
        color("silver")
        translate([20, -25, 100])
        rotate([0, -40, 45])
        scale([1, 1, 1.5])
        torus(r1=25, r2=chain_thickness/2);
    }
}

// --- FINAL ASSEMBLY ---

module hydrant() {
    color("red") {
        base_flange();
        main_body();
        side_nozzle(1);  // Right
        side_nozzle(-1); // Left
        front_boss();
        dome_cap();
    }
    chain_pair();
}

// Render the model
hydrant();
